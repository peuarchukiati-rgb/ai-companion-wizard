## The Idea

Most people use `.md` files as documentation тАФ something humans write and read.

This reverses that.

Here, `.md` is an **instruction set** тАФ something AI reads and acts on.

You're not reading this file.
Claude is.

And Claude behaves differently because of it.

This is what I call **Operation-Level Handoff** тАФ
a way to brief an AI agent completely,
the same way a mission brief activates a team.

No app. No API. No code.
Just a file. And a minion that gets to work.
